# 卡牌系统参数文档

## 97 概述

本文档定义了《Taberna Noctis》中所有卡牌的完整参数数据，包括材料卡牌和鸡尾酒卡牌的属性、效果和经济数值。

## 93 核心设计理念

- **状态对应**：每种材料对 5 种顾客状态（忙碌、急躁、烦闷、挑剔、友好）有不同的心情影响
- **数值平衡**：通过正负值平衡，确保没有万能材料
- **经济系统**：材料有采购成本，鸡尾酒有制作成本和售价
- **策略深度**：玩家需要根据顾客状态选择最优的材料组合

## 90 经济系统参数

### 基础数值

| 参数名称     | 数值 | 说明                         |
| ------------ | ---- | ---------------------------- |
| 初始金钱     | 500  | 游戏开始时的资金             |
| 顾客基础付费 | 30   | 每位顾客的固定付费           |
| 小费倍率     | 1.2  | 心情增量 ≥1 时的小费计算倍率 |

### 身份倍率

| 身份       | 倍率 | 说明         |
| ---------- | ---- | ------------ |
| 公司职员   | 1.0  | 基础倍率     |
| 小领导     | 1.2  | 中等消费能力 |
| 自由职业者 | 1.1  | 略高于基础   |
| 老板       | 1.5  | 最高消费能力 |
| 大学生     | 0.9  | 学生优惠     |

### 结算公式

```
顾客最终心情 = 初始心情 + ΔM（材料效果总和）
付费金额 = 饮品售价 + max(0, ΔM × 1.2 × 身份倍率)
```

**注意**：如果 ΔM 为负值，不会减少总付费金额，只是没有小费。

## 94 效果计算公式

### 心情增量计算

```
每材料Sequence值 = 强度 × 顺序权重 × 份量权重
最终心情增量ΔM = Σ(所有材料的Sequence值)
顾客最终心情 = 初始心情 + ΔM
```

### 付费计算

```
基础付费 = 30（固定）
小费 = max(0, ΔM × 1.2 × 身份倍率)
总付费 = 基础付费 + 小费
```

**注意**：

- 小费只在 ΔM > 0 时计算
- 如果 ΔM ≤ 0，顾客只支付基础费用 30，无小费
- 身份倍率影响小费数额

## 94 策略建议

### 高收益策略

1. **针对老板（1.5 倍率）**：

   - 推荐：尼格罗尼（挑剔+5）、曼哈顿（忙碌+3，挑剔+4）
   - 预期收益：基础 30 + 高心情增量 ×1.5 倍率

2. **针对挑剔顾客**：

   - 最优材料：金酒、干味美思、甜味美思、金巴利
   - 最优鸡尾酒：尼格罗尼(+5)、马天尼(+4)、曼哈顿(+4)

3. **安全选择（友好顾客）**：
   - 推荐：自由古巴(+4)、戴基里(+3)、古典(+2)
   - 风险低，稳定收益

### 风险管理

**避免负效果组合**：

- 忙碌顾客 + 莫斯科骡子(-5) = 严重心情下降
- 急躁顾客 + 玛格丽特(-4) = 大幅负面影响
- 挑剔顾客 + 自由古巴(-2) = 不满意
